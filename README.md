# Progetti di Tecnologie Web
### Esame di Scienze del testo per le professioni digitali - A.A. 2024/2025
Qui sono raccolti i progetti realizzati per l'esame di **Tecnologie Web**. L'obiettivo del corso era apprendere le basi dello sviluppo front-end, con un focus particolare sulla semantica HTML5, la stilizzazione CSS3 e il responsive design.

##Contenuto della Repository
La cartella contiene diversi esempi di siti web statici:

* **Portfolio**: Un sito che rappresenta me, una sorta di spazio che mi rappresenta.
* **Mandarì**: Landing page per Mandarì, un'azienda agrumicolo fittizia con design responsive.
* **Esercitazioni minori**: Piccoli frammenti di codice.
  
##Tecnologie Utilizzate
* **HTML5**: Utilizzo di tag semantici (`<article>`, `<section>`, `<nav>`) per l'accessibilità e la SEO.
* **CSS3**: 
    * Layout moderni tramite **Flexbox** e **CSS Grid**.
    * Design **Responsive** tramite Media Queries.
    * Variabili CSS per una gestione coerente dei colori.
* **Google Fonts & FontAwesome**: Per la tipografia e le icone.
* * **Bootstrap**: Per caroselli e oggetti responsive.
